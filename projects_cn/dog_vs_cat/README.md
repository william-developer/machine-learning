# 猫狗识别微信公众号应用


## 1.软件
* keras
* tesorflow
* numpy
* PIL
* web

## 2.数据集和模型训练
* 训练集：https://www.kaggle.com/c/dogs-vs-cats/data
* 验证集：http://www.robots.ox.ac.uk/~vgg/data/pets/data/images.tar.gz
* 模型训练时间：大约25小时，同一数据集上的准确率在91.34%，在验证集上的准确率在89.65%
