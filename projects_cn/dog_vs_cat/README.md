# 猫狗识别微信公众号应用


## 1.软件
* keras
* tesorflow
* numpy
* PIL
* web

## 2.数据集和模型训练
* 训练集：https://www.kaggle.com/c/dogs-vs-cats/data
* 模型训练时间：大约34小时，在验证集上的准确率在92.820%
* 模型优化之后，结合Resnet50,Xeption模型的特征，准确率提升到99.76%
